# tsparser-j2v8

`tsparser-j2v8` is an experiment to use [J2V8][j2v8] (Java wrapper around Google V8
Javascript engine) for parsing Typescript code (`.ts.` and `.tsx` files).

Please read [this post][post] on the experiment.
